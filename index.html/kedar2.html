    CSS

    In CSS first we select elements using tag, id or class.

    Ex:
    <p id="p">Paragraph</p> → use #
    <p class="p">Paragraph</p> → use .

    Then we apply styles inside curly brackets.

    #p { font-size: 20px; color: blue; }
    .p { font-size: 20px; color: blue; }

    CSS uses units like px, rem etc.

    Font-size: 1rem;

    Used to change text size.

    Color: blue;

    Used to change text color.

    Background-color: red;

    Used for background color.

    Height / Width

    Used to set size of elements.

    Text-align: center;

    Align text position.

    Border & Border-radius

    Used for borders and rounded corners.

    Display: flex;

    Used for layout.

    Position & Float

    Used for positioning elements.
