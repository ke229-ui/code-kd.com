    Easy boilerplate → type ! and press enter

    Every HTML element has opening & closing tags

    <h1>Heading</h1>

    h tag is used for headings. Bigger number = smaller size.

    <p>Paragraph</p>

    p tag is used for writing paragraphs.

    <button>Click</button>

    button tag is used to add buttons.

    <u>text</u>

    u tag is used to underline text.

    <ul><li>item</li></ul>

    ul tag creates unordered list.

    <ol><li>item</li></ol>

    ol tag creates ordered list.

    <li>item</li>

    li tag stores list items.

    <img src="image.jpg">

    img tag is used to add images.

    <table>...</table>

    table → table, tr → row, th → heading, td → data.

    <form></form>

    form tag is used to create forms.

    <input>

    input tag is used for user input.
