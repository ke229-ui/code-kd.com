    What is JS?
    JS is a programming language used to add logic to webpages.

    Comment: /*...*/ or // for one line.

    Data Types:
    Primitive → Number, String, Boolean, Undefined, Null, BigInt, Symbol

    Arithmetic operations: +, -, *, /
    Comparison: ==, ===, !==

    Conditional Statements:
    if (a === "blue") { color = "black"; }
    else { color = "white"; }
    else if (a === "blue") { color = "pink"; }

    Ternary Operator:
    age > 18 ? "adult" : "not adult";

    Loops:
    for (let i = 1; i < 5; i++) {
    console.log("college");
    }

    String Methods:
    toUpperCase() → GOD
    toLowerCase() → god

    Array:
    let heroes = ["Ironman", "Hulk", "Thor"];

    Important:
    document.getElementById("name")
    document.getElementsByClassName("name")
    document.getElementsByTagName("p")
